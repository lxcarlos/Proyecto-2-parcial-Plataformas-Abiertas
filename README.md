# Proyecto_2do_Parcial_LOGIN

Aplicación web desarrollada en equipo como proyecto del 2do parcial. Consiste en un sistema de login con 3 páginas relacionadas entre sí: login, perfil y explorar publicaciones.

## Tecnologías usadas

- HTML
- CSS
- JavaScript

## Páginas

- **login.html** — formulario de inicio de sesión con validación
- **perfil.html** — página de perfil con modo oscuro/claro y publicaciones
- **explorar.html** — buscador de publicaciones

## Funcionalidades

- Validación de usuario y contraseña
- Agregar y mostrar publicaciones en pantalla
- Filtrar publicaciones por texto
- Cambio de modo oscuro a claro
